# Franguitos — Site Institucional (v2, header compacto)

Alterações desta versão:
- Header **fixo, compacto e flutuante** (não ocupa a largura inteira; visual em formato de *pill*).
- Desktop com **poucos itens** (Produtos, Contato e WhatsApp).
- **Menu hamburger no mobile** com todas as seções e links (Produtos, Histórico, Contato, WhatsApp e Instagram).
- Responsividade refinada (grades 3→2→1 colunas, cartões e tipografia).
- Rodapé com links rápidos e CTA para WhatsApp.

## Como editar rapidamente
- **WhatsApp:** altere o número no `data-whatsapp` da tag `<body>` (DDI+DDD+número).
- **Instagram:** altere `data-instagram` da `<body>` se necessário.
